# Application Design Center

## 2025-12-05

### Feature

You can import Terraform modules to create components [(Preview)](https://cloud.google.com/products#product-launch-stages). For more information, see [Import components](https://docs.cloud.google.com/application-design-center/docs/import-components).

### Feature

You can set up Application Design Center using a single project boundary. For more information, see [Set up Application Design Center](https://docs.cloud.google.com/application-design-center/docs/setup).

### Feature

Application Design Center [supports](https://docs.cloud.google.com/application-design-center/docs/supported-resources) the following components:

* Google Kubernetes Engine (GKE) Node Pool
* GKE Standard Cluster
* GKE Autopilot Cluster
* Firestore

### Feature

You can export Terraform to your code repository. For more information, see [Export Terraform code](https://docs.cloud.google.com/application-design-center/docs/download-and-deploy#export_terraform_code).

### Announcement

Application Design Center is generally available [(GA)](https://cloud.google.com/products#product-launch-stages), including the following:

* [Application Design Center Google Cloud CLI](https://docs.cloud.google.com/sdk/gcloud/reference/design-center).
* [Design Center API](https://docs.cloud.google.com/application-design-center/docs/reference/rest).
