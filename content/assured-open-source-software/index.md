# Assured Open Source Software

## 2026-06-25

### Change

The Assured Open Source Software free tier includes an updated list of curated
binaries for the Go, Java, and Python languages.

For more information, see [List of supported Java and Python packages for the
Assured OSS free
tier](https://docs.cloud.google.com/assured-open-source-software/docs/supported-packages) and [List of supported
Go packages for the Assured OSS free
tier](https://docs.cloud.google.com/assured-open-source-software/docs/supported-packages-go).

---
## 2025-09-11

### Feature

Assured Open Source Software (Assured OSS) now supports Go packages. For more information, see [Download Go packages using direct repository access for the free tier](https://cloud.google.com/assured-open-source-software/docs/download-go-packages).
