# Cloud API Registry

## 2026-04-30

### Deprecated

Support for Model Context Protocol (MCP) servers and tools is deprecated. As of
the July 30, 2026 shutdown date, you can no longer retrieve, list, enable,
and disable MCP servers and tools using the Cloud API Registry
API. For more information, see
[Feature deprecations](https://docs.cloud.google.com/api-registry/docs/deprecations).

---
## 2025-12-10

### Announcement

Cloud API Registry is available in
[Preview](https://cloud.google.com/products/#product-launch-stages).

Cloud API Registry lets you discover, govern, use, and monitor
Model Context Protocol (MCP) servers and tools provided by Google, or by your
organization through Apigee API hub. For more information, see the
[Cloud API Registry overview](https://docs.cloud.google.com/api-registry/docs/overview).
