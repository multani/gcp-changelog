# Cloud Shell

## 2026-06-20

### Change

Cloud Shell no longer includes the Terraform CLI by default. Users can
[customize their environment](https://docs.cloud.google.com/shell/docs/configuring-cloud-shell#environment_customization)
to install the CLI on environment startup, or
install the binary to their home directory to persist the install between
sessions.
